# Fake News Classification Using LSTM

## 📌 Overview of the Project
Classification of news into categories like FAKE or REAL with the help of NLP and deep learning algorithms.

## 🧠 Technologies Used
- Python
- TensorFlow/Keras
- NLP
- LSTM (Long Short-Term Memory)

## 📂 Data Set
Taken data set from Kaggle Fake news data set.

## ⚙️ Model Structure
- Embedding layer
- LSTM layer (100 units)
- Dropout layers (to prevent overfitting)
- Dense layer (sigmoid activation)

## 🚀 Project Highlights
- Text pre-processing
- One-hot encoding
- Sequence padding
- Deep learning model training
- Binary classification

## 📊 Outcome
The model is built using binary cross entropy loss function.
